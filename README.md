# Genesis Card 💳

## Description
Genesis Card is a modern expense management and receipt tracking application designed to help users efficiently manage their financial transactions. It provides a clean and intuitive interface where users can add expenses, categorize them, and view summaries in real-time. The application is built using React and follows a modular component-based architecture, ensuring scalability and maintainability. Genesis Card aims to simplify personal finance tracking by offering a seamless user experience with responsive design and interactive components. Users can easily input expense details, view lists of transactions, and generate receipts for better financial organization. Whether you are managing daily expenses or tracking business costs, Genesis Card provides a reliable and structured solution. Its lightweight design ensures fast performance while maintaining clarity and usability. The project is ideal for learning React fundamentals and understanding how to build structured front-end applications.

## Features
- Add and manage expenses
- View expense list
- Generate receipts
- Clean and responsive UI
- Component-based architecture
- Easy-to-use interface

## Technologies Used
- React.js
- JavaScript (ES6)
- HTML5
- CSS3
- Node.js (for package management)
